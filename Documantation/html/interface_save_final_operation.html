Класс SaveFinalOperation

Очередь для сохранения финальных картинок в CoreDataShopCart.
Вызывается из менеджера SaveImageManager.
Пользователь могу редактировать картинки во время сохранения, чтобы не перегружать CoreData мы сохраняем данные возможно измененные пользователем еще раз перед переходом в корзину. Подробнее...

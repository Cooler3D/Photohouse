Подробное описание

Класс конструктора альбомов

Данный класс используется для парсинга данных от сервера, а так же манипуляции с разворотами констрктора Может содержать пользовательский шаблон, который требуется синхронизировать с CoreDataShopCart.
Сожержатся внутри развороты Layout

Автор
Дмитрий мартынов
Предупреждения
для отображения на экране используйте класс PageView

 CPrintDataПокупка пользователя.
Представляет собой сфорированную покупку для отправки на сервер PhotoHouse Вы можете добавлять картинки, удалять, Менять параметры.
Также менять цвет, размер, обложку, и др параметры иесли у них есть выбор.
Вместе с эти объектом используется:
StoreItem - идетификатор покупки, при выборе в магазине StoreViewController, хранятся в CoreDataStore
Template - шаблон для конструктора альбомов
